# Useful Prompts

A small prompt library for reusable LLM system prompts, coaching templates, and other prompt patterns.

## What this project is

This repository stores prompt files organized by topic or use case. Each prompt is kept in a plain text file so it can be reused, edited, and shared easily.

## Structure

- `general/` — general-purpose prompts and coaching/system prompt templates
- other folders may be added over time for more categories

## Prompts in this library

- `general/life_coach.txt` — a Life Coach system prompt designed to help an LLM act as a structured, practical, and supportive mentor for personal growth.

## How to use

1. Open a prompt file.
2. Copy the contents into your LLM prompt field or system prompt.
3. Adapt the prompt to your specific situation as needed.

## Contributing

- Add new prompt files in clearly named folders.
- Keep each file focused on a single prompt or prompt pattern.
- Use descriptive filenames and include a short header or comments if helpful.

## Notes

This repo is designed to grow over time with more prompts for work, productivity, creativity, coaching, and more.
